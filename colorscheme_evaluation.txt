Most tests (starting from blackbeauty) have been done in a mostly vanilla vim
environment (i.e. closing and re-opening vim for each test). The reason for
this is that loading one colorscheme might still keep colors from the previous
one so the results would be non-representative for someone starting vim "from
scratch". No colorscheme was set, and "background" was set to the appropriate
value (light or dark)

N: Not applicable
-: Worse than the default colorscheme
+: Improvement on the default colorscheme
o: Not much (or no) change from default colorscheme
